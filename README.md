Instructiuni de utilizare:
Pentru a folosi acest script, introduceti in mnt.conf tot path-ul spre directorul ehd din fisier(de exemeplu:/home/nume/proiect/AutomounterShell/ehd), apasati space si introduceti device-ul extern pe care vreti sa il montati(de exemplu:/dev/sda5).
Rulati cu ./amsh.sh si scrieti comanda "cd ehd" si se va monta device-ul extern in directorul ehd.
Pentru a iesi din script scrieti in command prompt "q" sau "exit".
Demontarea fisierelor se va face automat la un interval de 5 minute de inactivitate.
